## Schematic

| Path | Purpose |
|:--|:--|
| `schematic/pt100_readout_schematic.pdf` | Buildable schematic: improved Howland 1.000 mA current source, 4-wire PT100, INA128 instrumentation amp (×10), difference amplifier (×7.87) with a shared-reference baseline. Full BOM and net list in the sheet. |

## SPICE

| Path | Purpose |
|:--|:--|
| `spice/ideal_models.lib` | Ideal macromodels used for DC validation. `INA128_IDEAL` is a fixed-gain (10.107) instrumentation amp referred to its REF pin; `OPAMP_IDEAL` is an open-loop VCVS op-amp. Every port is tied to ground through 1 GΩ so no node floats. |
| `spice/vout_sweep_4wire.csv` | ngspice `.dc` sweep of the sensor resistance 100 → 138.5 Ω (0 → 100 °C), 101 points. Columns: RPT (Ω), V(Vout) (V). 4-wire sensing. |
| `spice/vout_sweep_2wire.csv` | Same sweep with a 2 Ω lead resistance (R5) inside the sensed loop and the INA sensing on the electronics side of it. 2-wire sensing. |

## Source files

| Path | Purpose | LoC |
|:--|:--|:--|
| `experiments/recovery_analysis.py` | Loads the 4-wire sweep, fits the readout transfer, inverts each output through the Callendar–Van Dusen relation to recover temperature, and quantifies the residual for the ideal analog chain and for a 12-bit ADC. Generates `plots/recovery_result.png`. | ~30 |
| `experiments/lead_study.py` | Compares the 4-wire and 2-wire SPICE sweeps under a common calibration and reports the apparent-temperature error introduced by the lead resistance. Generates `plots/lead_resistance_result.png`. | ~35 |

## KiCad source

The `.kicad_sch` project is maintained locally and is not yet committed. The two schematic PDFs above are exact renders of the buildable and simulation-ready sheets; the KiCad source can be added once the project is de-versioned for public release.

## Reproducibility

Python 3.10+, `pip install -r requirements.txt`. From the repository root:

```
python experiments/recovery_analysis.py
python experiments/lead_study.py
```

Both scripts read the committed CSVs in `spice/` and write their figures to `plots/`. The SPICE sweeps themselves were produced in KiCad 8 / ngspice from the simulation-ready schematic with `.dc RPT 100 138.5 0.385`.
